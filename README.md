# RSS Feed Reader 

## :white_check_mark: General info

RSS Reader with automatic updating.

http://helen-rss.surge.sh


## :arrow_forward: How to Run App

1. cd to the repo
2. Setup
  - run `make install`
2. Run Build
  - run `make develop`

## :hammer: Technologies

Project is created with:
* ES6
* AJAX using Axios
* Node.js
* JQuery
* Webpack
* DOMParser
